# Roadmap: MovieRecom UI Improvements

## Overview

Enhance user experience with a dedicated "Liked Movies" page showing all rated films and improved profile page design. Focus on clean, intuitive UI that follows existing dark theme aesthetics.

## Domain Expertise

None (ASP.NET MVC, Razor, Bootstrap already in stack)

## Phases

- [ ] **Phase 1: Liked Movies Page** - New page for managing rated movies
- [ ] **Phase 2: Profile Page Redesign** - Improve profile UI/UX

## Phase Details

### Phase 1: Liked Movies Page
**Goal**: Create /Movies/Liked page showing all movies user has rated with ability to remove ratings
**Depends on**: Nothing (first phase)
**Research**: Unlikely (using existing MVC patterns)
**Plans**: 2 plans

Plans:
- [ ] 01-01: Backend - Controller action and view model for liked movies
- [ ] 01-02: Frontend - Razor view with cards, filters, and remove rating functionality

### Phase 2: Profile Page Redesign
**Goal**: Improve /Account/Profile page UI with better layout, styling, and user experience
**Depends on**: Nothing (independent from Phase 1)
**Research**: Unlikely (internal UI patterns)
**Plans**: 2 plans

Plans:
- [ ] 02-01: Profile layout restructure - modern card-based design
- [ ] 02-02: Profile enhancements - avatar upload, stats, activity timeline
